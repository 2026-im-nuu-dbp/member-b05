## 快速開始指南

### 1. 資料庫設置

1. 打開 phpMyAdmin
2. 創建一個新的資料庫，名稱為 `test_db`
3. 進入新創建的資料庫
4. 點擊 "匯入" 標籤
5. 選擇 `news.sql` 文件並匯入
6. 確認所有表格已成功創建

### 2. 檔案配置

- 確保 `db_config.php` 中的資料庫連接信息正確
- 默認配置：
  - 主機: localhost
  - 資料庫: test_db
  - 用戶: root
  - 密碼: (空白)

### 3. 訪問應用

1. 打開瀏覽器
2. 訪問 `http://localhost/week11-0506-hw7/member-b05/`
3. 選擇 "註冊" 創建新帳號
4. 完成首次登入設置

### 4. 建立管理員帳號

第一個註冊的帳號需要手動設置為管理員：

```sql
UPDATE members SET is_admin = 1 WHERE id = 1;
```

或通過其他管理員帳號在管理面板中設置。

### 5. 使用功能

#### 普通用戶
- 註冊和登入
- 查看討論
- 發表和回應討論
- 編輯個人檔案

#### 管理員
- 管理會員（設置/移除權限、刪除）
- 管理分類（新增/刪除）
- 查看系統統計信息

### 常見問題

**Q: 出現"資料庫連線失敗"**
A: 檢查 db_config.php 的連接參數是否正確，確保 MySQL 服務正在運行。

**Q: 忘記了密碼**
A: 管理員可以在管理面板中刪除該用戶，用戶可重新註冊。

**Q: 第一次登入後無法發表討論**
A: 確保首次登入的檔案設置已完成（設置暱稱、頭像、顏色）。
